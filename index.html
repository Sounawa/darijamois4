📖 Les expressions conditionnelles et le subjonctif

Conditionnel avec "ila" (si) :
ila jiti, nfer7 (si tu viens, je serai content).

Subjonctif pour exprimer le souhait ou l'obligation :
On utilise la particule "bghi" (vouloir) + verbe au présent.
bghit nemshi (je veux aller).

Impératif :
Pour le "tu" : radical du verbe (ex: ktb! écris !)
Pour "vous" : ajouter -u (ex: ktbu! écrivez !).
